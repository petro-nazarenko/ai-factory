## [0.1.0] - 2026-03-25

### Added

- Initial project structure with `pyproject.toml`
- Base integration framework (`src/integrations/base.py`)
- Google Sheets integration (`src/integrations/google_sheets.py`)
- PDF processing module (`src/integrations/pdf_processor.py`)
- Email handler module (`src/integrations/email_handler.py`)
- CLI with Typer (`src/cli.py`)
- Utility modules (logger, config, retry)
- Comprehensive test suite (`tests/`)
- Example: Bol.com API integration (`examples/bol_com/`)
- Example: PDF to Sheets workflow (`examples/pdf_to_sheets.py`)
- Example: Email automation (`examples/auto_email.py`)
- MkDocs documentation structure (`docs/`)
- GitHub Actions CI/CD workflows
- Pre-commit hooks configuration
